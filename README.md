## Overview

This project was built end-to-end by one person over the course of a month, combining 3D modeling, game engine development, UI design, and sound design into a single playable experience.

## Process

1. **Modeling** — Built the Ford Mustang GT 2024 from scratch in Blender, learning vehicle topology and surfacing along the way.
2. **Engine Import** — Brought the car into Unreal Engine 5, set up correct scale, and configured collisions.
3. **Drivability** — Built out the Blueprint logic for vehicle physics, input controls, and handling.
4. **Track Design** — Modeled a custom race track in Blender and imported it into UE5 as the driving environment.
5. **UI/UX** — Designed a speedometer HUD in Figma and implemented it as in-game UI.
6. **Audio** — Recorded/sourced and integrated engine startup, idle, gear shift, and rev sounds tied to vehicle state.
7. **Lighting** — Set up an HDRI environment for realistic lighting and reflections.
